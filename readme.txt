KD Coming Soon - responsive and fluit landing page for Wordpress web sites. Use it as a Coming Sonn or Under Maintenance page for your site.

== Description ==

KD Coming Soon is a landing page for Wordpress web sites. Use it as a Coming Soon or Under Maintenance page for your site.
Included is a subscription functionality which allows user to signup and be notified once the site go live. Admin/owner get sent email copy of all subscriptions.
Owner can edit and control various aspects of the page such as add/remove header slides, add/remove social links, edit text and more...

=Features  =

* Replace your website with a coming soon or under maintenance page.
* Fully responsive and fluit for all platforms.
* Full edit control of page.
* Adjust the timings at your wish.
* Add as many slides you wish.
* Link to all your social-media platforms.
* Collect users by let them subscribe.
* Above all, Free to use.

== Installation ==

If you have ever installed a WordPress plugin, then installation will be pretty easy:

1. Upload 'kd-coming-soon' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In wordpress backend go into KD Coming Soon from within the admin main menu.
4. Set page content text / messages.
5. Set the time the KD Coming Soon is active.
6. Add/Delete slides by clicking add new/delete button.
7. Switch KD Coming Soon on.
8. Hit Save Now and you're done!
